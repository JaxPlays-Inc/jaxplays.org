## Artist Profile Application { class="f3 mt4 mb3 sectionline" }

Apply to be considered for the JaxPlays People Directory. Whether you're an established theatre professional or an emerging artist, this is the place to request a public profile in Jacksonville's live theatre record.

Profiles are reviewed before publication so the directory stays accurate, useful, and connected to the regional theatre community.

{{< yellow-button href="/apply/profile/" >}}Apply for an Artist Profile{{< /yellow-button >}}

## Production Listing Application { class="f3 mt4 mb3 sectionline" }

Apply to have an upcoming or past production considered for the JaxPlays production directory and calendar. This helps audiences, artists, and theatre makers find accurate show information in one trusted place.

Production listings are reviewed before publication so the archive remains dependable.

{{< yellow-button href="/apply/production/">}}Apply for a Production Listing{{< /yellow-button >}}

## Audition Announcement Application { class="f3 mt4 mb3 sectionline" }

Apply to have an audition announcement considered for publication on JaxPlays. Share the essential details so performers, musicians, technicians, and creative collaborators can find the opportunity quickly.

{{< yellow-button href="/apply/audition/">}}Apply to Post an Audition{{< /yellow-button >}}

## Theatre Listing Application { class="f3 mt4 mb3 sectionline" }

Apply to have your theatre company considered for the JaxPlays theatre directory. The directory connects audiences, artists, and organizations across Northeast Florida and Southeast Georgia.

Theatre listings are reviewed before publication so JaxPlays can keep the record clear, current, and useful.

{{< yellow-button href="/apply/theatre/">}}Apply for a Theatre Listing{{< /yellow-button >}}
